## Gemini / Google

```sh
  # see: https://github.com/google-gemini/gemini-cli
  npm install -g @google/gemini-cli
  # Get your key from https://aistudio.google.com/apikey
  export GEMINI_API_KEY="YOUR_API_KEY"
  gemini
```
